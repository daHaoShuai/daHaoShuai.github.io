1、Git介绍

Git（读音为/gɪt/）是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本 管理。

Git 是 Linus Torvalds 为了帮助管理 Linux内核开发而开发的一个开放源码的版本控制软件。

大概知道它是个开源的分布式版本控制系统就行了，我们主要学习Git的使用。
