# 开始学习

TIP

安卓（Android）是一种基于内核（不包含组件）的自由及开放源代码的操作系统。主要使用于移动设备，如智能手机和平板电脑，由美国Google公司和开放手机联盟领导及开发。Android操作系统最初由Andy Rubin. Rubin)开发，主要支持手机。2005年8月由Google收购注资。2007年11月，Google与84家硬件制造商、软件开发商及电信营运商组建开放手机联盟共同研发改良Android系统。随后Google以Apache开源许可证的授权方式，发布了Android的源代码。第一部Android智能手机发布于2008年10月。Android逐渐扩展到平板电脑及其他领域上，如电视、数码相机、游戏机、智能手表等。2011年第一季度，Android在全球的市场份额首次超过塞班系统，跃居全球第一。 2013年的第四季度，Android平台手机的全球市场份额已经达到78.1%。2013年09月24日谷歌开发的操作系统Android在迎来了5岁生日，全世界采用这款系统的设备数量已经达到10亿台。

最后重新安装

一些快捷键

补全代码：Ctrl+Alt+空格

格式化代码：Ctrl+Alt+L

Android Studio 一些选项

Make Project:

编译Project下所有Module，一般是自上次编译后Project下有更新的文件，增量编译，不生成Apk。

Make Selected Modules:

编译指定的Module，一般是自上次编译后Module下有更新的文件，增量编译，不生成Apk。

Clean Project:

删除之前编译后的编译文件。部分版本的AS会自动重新编译整个Project，不生成Apk。

Rebuild Project:

先执行Clean操作，删除之前编译的编译文件和可执行文件，然后重新编译新的编译文件，不生成Apk

Build APK:

前面4个选项都是编译，没有生成apk文件[旧版本AS，新版本的AS会自动生成debug.apk]，如果想生成Apk，需要点击Build APK。

Generate Signed APK:

生成有签名的Apk(一般项目嵌入第三方，生成release包时必须混淆，否则无法生成Apk)。

# 线性布局LinearLayout

该布局中所有的视图都是以一条直线方向来进行布局的。

方向：垂直（Vertical）、水平（Horizontal）

所以在每次使用线性布局的时候，一定是需要指定方向的。（默认是水平的）。

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">
</LinearLayout>

常用的属性

android:orientation="vertical"：线性布局的方向  垂直(vertical)/水平(Horizontal)
android:id="@+id/ID名称"：控件/布局的ID属性

android:background="@color/colorGreen"：背景颜色，资源的形式表示颜色。
android:background="#f00"：背景颜色，是以#开头的十六进制表示颜色
android:background="#ff0000"：背景颜色，是以#开头的十六进制表示颜色
R    G  B
FF    00   00
00    FF   00
00   00    FF
https://www.cnblogs.com/stalla/p/11593566.html

android:layout_width=""：宽度
android:layout_height=""：高度
1.match_parent（匹配父窗口）
2.wrap_content（适应内容）
3.xxxdp(直接写具体的值，单位是dp)

android:text="今天下雨啦啦啦"：显示的文本（一种是使用资源的形式描述，一种是直接卸载布局文件中）
android:textSize="20sp"：文字的大小（单位是sp）
android:textColor="#00f"：字体的颜色（跟背景颜色的写法是一致的）
android:textStyle="bold|italic"：字体风格，（加粗、斜体、正常）
android:gravity="bottom|right"：内容的位置“上下左右”
android:layout_gravity="bottom":控件在布局中的位置

android:weightSum="10"权重和
android:layout_weight="5"权重比例
如果方向是垂直的，高度设置为0dp。
如果方向是水平的，宽度设置为0dp。

WARNING

内外边距

控件和控件之间的距离称为外边距。

控件中的内容与控件之间的距离称为内边距。

android:layout_margin="100dp"：外边距（注意四个方向都是有效果的，单位是dp）

android:layout_marginLeft="50dp"：左外边距

android:layout_marginTop="10dp"：上外边距

android:layout_marginBottom="50dp"：下外边距

android:layout_marginRight="100dp"：右外边距

android:padding="10dp"：内边距（注意四个方向都是有效果的，单位是dp）

android:paddingLeft="90dp"：左内边距

android:paddingTop="150dp"：上内边距

android:paddingBottom="100dp"：下内边距

android:paddingRight="100dp"：右内边距

# ImageView图片视图

WARNING

在使用ImageView组件显示图像时，通常需要将显示的图像放置在res/drawable或者res/mipmap目录下即可。

android:scaleType=""：用于设置所显示图片如何缩放或移动以适应ImageView的大小，属性值可以是：

1.matrix：表示进行缩放

2.fitXY：图片的横向，纵向独立缩放，使得该图片完全适应于ImageView的大小。

3.fitStart：图片的纵横比缩放图片，直到该图片显示在ImageView中，缩放完成后图像在ImageView的左上角。

4.fitCenter：图片的纵横比缩放图片，直到该图片显示在ImageView中，缩放完成后图像在ImageView的中间。

5.fitEnd：图片的纵横比缩放图片，直到该图片显示在ImageView中，缩放完成后图像在ImageView的右下角。

6.center：以图片的中心点和ImageView的中心为基准，按照图片的原来大小居中显示，不缩放，用ImageView的大小截取图片的居中部分。

7.centerCrop：使得图片完全覆盖ImageView的大小。

8.centerInside：完全显示该图片，

android:adjustViewBounds="true"：自动调整图片的边界（宽和高）

true：自动调整边界

false：默认值。不需要自动调整边界

public class MainActivity extends AppCompatActivity {

    RadioButton[] rb_sex;
    TextView textView;
    RadioGroup rg_grade;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        rb_sex = new RadioButton[3];
        rb_sex[0] = findViewById(R.id.male);
        rb_sex[1] = findViewById(R.id.female);
        rb_sex[2] = findViewById(R.id.unkown);
        textView = findViewById(R.id.tv);
        rg_grade = findViewById(R.id.grade);

    }

    public void btnGetData(View view) {
        //1.遍历每一个单选框，判断哪个的checked状态为true。
        String strSex = "";

            }

        }

        //2.获取对应的组，在直接获取组中选中的RadioButton。
        int checkId = rg_grade.getCheckedRadioButtonId();
        RadioButton button = findViewById(checkId);
        String strGrade = button.getText().toString();

        textView.setText("你选择的性别是:" + strSex + ",你选择年级是:" + strGrade);
    }
}

MainActivity.java

public class MainActivity extends AppCompatActivity {
    TextView textView;
    CheckBox[] cbs;


    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        cbs = new CheckBox[4];
        cbs[0] = findViewById(R.id.cb1);
        cbs[1] = findViewById(R.id.cb2);
        cbs[2] = findViewById(R.id.cb3);
        cbs[3] = findViewById(R.id.cb4);
        textView = findViewById(R.id.textView);
    }

    public void btnGet(View view) {
        List<String> listStr = new ArrayList<>();
        for (int i = 0; i < cbs.length; i++) {
            if (cbs[i].isChecked()) {
                listStr.add(cbs[i].getText().toString());
            }

        }
        String result = Arrays.toString(listStr.toArray());
        textView.setText(result);
    }
}

public class MainActivity extends AppCompatActivity {
    Spinner spinner;
    EditText editText;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        spinner = findViewById(R.id.spinner);
        editText = findViewById(R.id.ed);
        //下拉列表的监听事件
        spinner.setOnItemSelectedListener(new AdapterView.OnItemSelectedListener() {
            @Override
            public void onItemSelected(AdapterView<?> parent, View view, int position, long id) {
                String result = parent.getItemAtPosition(position).toString();
                editText.setText(result);
            }

            @Override
            public void onNothingSelected(AdapterView<?> parent) {

            }
        });

    }
}
