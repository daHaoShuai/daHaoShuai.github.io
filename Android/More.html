# 适配器

WARNING

在Android中,ListView中常用的适配器:

1.ArrayAdapter:数组适配器:一行显示一个控件,该控件必须是TextView

2.SimpleAdapter:简单适配器:一行显示多个控件,数据源是 List列表

3.BaseAdapter:基础适配器:一行显示多个控件

public class MyAdapter extends BaseAdapter {
    Context context;
    String[] fruitName;
    int[] imageName;

    public MyAdapter(Context context, String[] fruitName, int[] imageName) {
        this.context = context;
        this.fruitName = fruitName;
        this.imageName = imageName;
    }

    //绑定列表的子项数目
    @Override
    public int getCount() {
        return fruitName.length;
    }

    //根据索引获取该位置的对象
    @Override
    public Object getItem(int position) {
        return null;
    }

    //获取子项的ID
    @Override
    public long getItemId(int position) {
        return 0;
    }

    //获取子项要显示的界面
    @Override
    public View getView(int position, View view, ViewGroup parent) {
        LayoutInflater inflater = LayoutInflater.from(context);
        View itemView;
        ViewHolder viewHolder;
        if (view == null) {
            itemView = inflater.inflate(R.layout.item, parent, false);
            viewHolder = new ViewHolder();
            viewHolder.imageView = itemView.findViewById(R.id.itemImage);
            viewHolder.textView = itemView.findViewById(R.id.itemName);
            itemView.setTag(viewHolder);
        } else {
            itemView = view;
            viewHolder = (ViewHolder) itemView.getTag();
        }
        viewHolder.imageView.setImageResource(imageName[position]);
        viewHolder.textView.setText(fruitName[position]);
        return itemView;
    }

    class ViewHolder {
        ImageView imageView;
        TextView textView;
    }

}

public class MyAdapter extends RecyclerView.Adapter<MyAdapter.ViewHolder> {
    private List<Fruit> fruitList;

    public MyAdapter(List<Fruit> fruitList) {
        this.fruitList = fruitList;
    }

    @Override
    public MyAdapter.ViewHolder onCreateViewHolder(ViewGroup parent, int viewType) {
        View view = LayoutInflater.from(parent.getContext()).inflate(R.layout.item, parent, false);
        ViewHolder viewHolder = new ViewHolder(view);
        return viewHolder;
    }

    @Override
    public void onBindViewHolder(MyAdapter.ViewHolder holder, int position) {
        Fruit fruit = fruitList.get(position);
        holder.imageView.setImageResource(fruit.getImageId());
        holder.textView.setText(fruit.getFruitName());

    }

    @Override
    public int getItemCount() {
        return fruitList.size();
    }


    public class ViewHolder extends RecyclerView.ViewHolder {
        ImageView imageView;
        TextView textView;

        public ViewHolder(View itemView) {
            super(itemView);
            imageView = itemView.findViewById(R.id.itemImage);
            textView = itemView.findViewById(R.id.itemName);
        }
    }

}

# Activity活动

TIP

Intent意图

在各个组件之间来回通信和协助通信的组件(Activity,Server,BroadCastReceiver)

Intent分为两种:一种是明确的(显示)Intent,另一种是不明确(隐式)Intent.

明确的(显示)Intent:指定要启动的目标组件是哪个.

不明确的(隐式)Intent:没有指定要启动的目标组件,通过一系列的过滤条件隐式去触发需要启动的目标组件.

                    e.printStackTrace();
                }

        }

    public void load() {
        BufferedInputStream bis = null;

        try {
            bis = new BufferedInputStream(openFileInput("day1127.txt"));
            ByteArrayOutputStream baos = new ByteArrayOutputStream();
            byte[] buffer = new byte[1024];
            while (true) {
                int hasRead = bis.read(buffer);
                if (hasRead < 0) {
                    break;
                }
                baos.write(buffer, 0, hasRead);
            }
            editText.setText(baos.toString());
        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (bis != null) {
                try {
                    bis.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }

    }

        save();
    }

TIP

SDCard下目录的划分:

公有目录:指的是SDCard的九个公有目录,Environment类可以访问

私有目录:通过Context访问.

在Android6.0以上系统中Google 将权限分为两种:

1.普通权限:直接在配置文件中添加的.

2.危险权限:需要进行动态的申请.

如果需要对SDCard进行操作,最好讲普通权限和危险权限都需要申请.

1.声明要访问的权限和请求码.

public class SDCardUtils {
    //判断SDCard是否挂载
    public static boolean isSDCardMounted() {
        if (Environment.getExternalStorageState().equals(Environment.MEDIA_MOUNTED)) {
            return true;
        }
        return false;
    }


    // 获得SDCard的根目录 如/storage/sdcard/
    public static String getSDCardBaseDir() {
        //如果挂载成功,则输出绝对路径
        if (isSDCardMounted()) {
            return Environment.getExternalStorageDirectory().getAbsolutePath();
        }
        return null;
    }

    //获得SD卡的全部空间大小(单位:M)
    public static long getSDCardSize() {
        if (isSDCardMounted()) {
            StatFs statFs = new StatFs(getSDCardBaseDir());
            //最小SDK需要在18以上
            long blockCount = statFs.getBlockCountLong();
            long blockSize = statFs.getBlockSizeLong();
            return blockSize * blockCount / 1024 / 1024;

        }
        return 0;
    }

    //获取SDCard空闲空间的大小(单位:M)
    public static long getSDCardFreeSize() {
        if (isSDCardMounted()) {
            StatFs statFs = new StatFs(getSDCardBaseDir());
            //最小SDK需要在18以上
            long freeblockCount = statFs.getFreeBlocksLong();
            long blockSize = statFs.getBlockSizeLong();
            return blockSize * freeblockCount / 1024 / 1024;

        }
        return 0;
    }

    // 获取SDCard可用空间的大小(单位:M)
    public static long getSDCardAvailSize() {
        if (isSDCardMounted()) {
            StatFs statFs = new StatFs(getSDCardBaseDir());
            //最小SDK需要在18以上
            long avablockCount = statFs.getAvailableBlocksLong();
            long blockSize = statFs.getBlockSizeLong();
            return blockSize * avablockCount / 1024 / 1024;

        }
        return 0;
    }

    //往SDCard公有目录下保存文件 (九大公有目录中的一个，具体由type指定) /storage/sdcard/{type}
    public static boolean saveDataSDCardPublicDir(byte[] data, String type, String filename) {
        if (isSDCardMounted()) {
            BufferedOutputStream bos = null;
            File file = new File(Environment.getExternalStorageDirectory() + File.separator + type);
            Log.i("SDCard", file + "");
            try {
                bos = new BufferedOutputStream(new FileOutputStream(new File(file, filename)));
                bos.write(data);
                bos.flush();

            } catch (FileNotFoundException e) {
                e.printStackTrace();
            } catch (IOException e) {
                e.printStackTrace();
            } finally {
                if (bos != null) {
                    try {
                        bos.flush();
                    } catch (IOException e) {
                        e.printStackTrace();
                    }

                }
            }

        }
        return true;
    }

                    }

                }
            }

            BufferedOutputStream bos = null;
            File file = context.getExternalCacheDir();
            Log.i("SDCard", file + "");
            try {
                bos = new BufferedOutputStream(new FileOutputStream(new File(file, filename)));
                if (filename.endsWith(".png") || filename.endsWith(".PNG")) {
                    bitmap.compress(Bitmap.CompressFormat.PNG, 100, bos);
                } else if (filename.endsWith(".jpg") || filename.endsWith(".JPG")) {
                    bitmap.compress(Bitmap.CompressFormat.JPEG, 100, bos);
                }

                bos.flush();

            } catch (FileNotFoundException e) {
                e.printStackTrace();
            } catch (IOException e) {
                e.printStackTrace();
            } finally {
                if (bos != null) {
                    try {
                        bos.flush();
                    } catch (IOException e) {
                        e.printStackTrace();
                    }

                }
            }

        }

        return true;
    }

    //从SDCard读取指定文件 /storage/sdcard/{filePath}
    public static byte[] loadFileFromSDCard(String filePath) {
        BufferedInputStream bis = null;

        try {
            File file = new File(getSDCardBaseDir());
            bis = new BufferedInputStream(new FileInputStream(new File(file, filePath)));
            ByteArrayOutputStream baos = new ByteArrayOutputStream();
            byte[] buffer = new byte[1024];
            while (true) {
                int hasRead = bis.read(buffer);
                if (hasRead < 0) {
                    break;
                }
                baos.write(buffer, 0, hasRead);
            }
            return baos.toByteArray();
        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (bis != null) {
                try {
                    bis.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }

        return null;
    }
