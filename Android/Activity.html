# MVC模型

TIP

M:(Module数据模型):用于获取和访问数据源(数据库\网络\传感器采集)

V:(View视图):用于给用户提供可视化的界面

C(Controller控制器):上述M和V之间的桥梁

TIP

Android四大组件:

Activity:活动,与视图进行交互的组件

Server:服务,在后台提供一定的服务,复杂的计算,不会跟界面产生直接的关系

ContentProvider:内容提供者.主要的是给程序提供数据

BroadCastReceiver:广播.应用程序内部或者应用程序之间的通信桥梁

# Activity活动

TIP

Intent意图

在各个组件之间来回通信和协助通信的组件(Activity,Server,BroadCastReceiver)

Intent分为两种:一种是明确的(显示)Intent,另一种是不明确(隐式)Intent.

明确的(显示)Intent:指定要启动的目标组件是哪个.

不明确的(隐式)Intent:没有指定要启动的目标组件,通过一系列的过滤条件隐式去触发需要启动的目标组件.

TIP

新建Activity

首先新建一个新的Activity需要注意的事项:最好一步到位去创建.确定一个Activity是否是完整有三个部分:

1.对应有*.java结尾文件

2.对应有*.xml结尾的文件

3.配置文件中有声明

# Activity的生命周期

WARNING

生命周期:从出生到死亡

Android的Activity生命周期共有7格方法:

onCreate():创建的时候被调用

onStart():启动的时候被调用

onResume():可见的时候被调用

onPause():不可见的时候被调用

onStop():停止的时候被调用

onDestory():销毁时候被调用

onRestart():重启的时候被调用

TIP

在应用程序启动的时候,依次调用:

​ onCreate--onStart--onResume方法

当程序退出时:

​ onPause--onStop--onDestory

TIP

Android中的日志:有5个级别:

Verbose 冗余信息

Debug 调试信息

Info 普通信息

Warn 警告信息

Error 错误信息

# Service

TIP

Service是四大组件之一,称为服务.

服务是什么?

业务(前台) 服务(后台)

什么东西适合放到Service中去做(服务)

1.与硬件进行交互的

2.启动子线程.进行耗时操作.网络操作等.

3.跨进程提供Service端的API(AIDL\Message)

启动Service的两种方式:

​ startService:启动

​ bindService:绑定

# startService

TIP

Service的启动先后触发**onCreate()--onStartCommand()--onDestroy()**方法.

如果需要启动一个服务,需要在上下文中调用:

context.startService(intent);
1

如果需要停止一个服务,需要在上下文中调用:

context.stopService(intent);
1

新建一个服务类:MyService.java

最好是一步到位:

MyService.java

public class MyService extends Service {
    public MyService() {
    }

    @Override
    public IBinder onBind(Intent intent) {
        // TODO: Return the communication channel to the service.
        return null;
    }

    //只有第一个startService()时才会调用
    @Override
    public void onCreate() {
        Log.i("MyService","onCreate");
        super.onCreate();
    }
    //每次startService()时都会调用
    @Override
    public int onStartCommand(Intent intent, int flags, int startId) {
        Log.i("MyService","onStartCommand");
        return super.onStartCommand(intent, flags, startId);
    }

    //stopservice时调用
    @Override
    public void onDestroy() {
        Log.i("MyService","onDestroy");
        super.onDestroy();
    }
}

TIP

        }
    };

# 数据存储概述

TIP

Android中存储数据的方式:

1.SharePreferences(保存最简单的键值对/data/data/包名/shared_pred目录)

2.文本存储(内部存储,外部存储)

2.1内部存储,存放在工程目录内部(data/data/包名/file/目录)

2.2外部存储.外部的SDCard/storage/sdcard/.....

3.网络存储(HTTP\Socket\TCP\UDP)

4.SQLite(Android设备内部数据库)

5.ContentProvider

# 1.SharePreferences

TIP

SharePreferences使用步骤:(保存最简单的键值对)

1.通过getSharePreferences()获取SharePreferences的对象

2.通过SharePreferences提供的edit()方法获取内部接口Editor对象

3.通过Editor的putXXX()方法写入值.

4.通过Editor的commit()方法提交.此时数据已经下载XXX文件中,当后面需要用的时候.

通过SharePreferences的getXX()方法可以读取数据.

                break;
            //读入数据
            case R.id.btnGet:

                editText2.setText(values);
                break;
            default:
                break;
        }
    }
}

public class MainActivity extends AppCompatActivity {
    EditText editText;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        editText = findViewById(R.id.edit3);
        load();
    }

    public void save() {
        BufferedOutputStream bos = null;
        try {
            bos = new BufferedOutputStream(openFileOutput("day1127.txt", Context.MODE_PRIVATE));
            bos.write(editText.getText().toString().getBytes());
            bos.flush();
        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (bos != null) {
                try {
                    bos.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }

            }
        }

    }

    public void load() {
        BufferedInputStream bis = null;

        try {
            bis = new BufferedInputStream(openFileInput("day1127.txt"));
            ByteArrayOutputStream baos = new ByteArrayOutputStream();
            byte[] buffer = new byte[1024];
            while (true) {
                int hasRead = bis.read(buffer);
                if (hasRead < 0) {
                    break;
                }
                baos.write(buffer, 0, hasRead);
            }
            editText.setText(baos.toString());
        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (bis != null) {
                try {
                    bis.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }

    }

    @Override
    protected void onDestroy() {
        super.onDestroy();
        save();
    }
}
