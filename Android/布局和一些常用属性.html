# 线性布局LinearLayout

TIP

该布局中所有的视图都是以一条直线方向来进行布局的。

方向：垂直（Vertical）、水平（Horizontal）

所以在每次使用线性布局的时候，一定是需要指定方向的。（默认是水平的）。

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">
</LinearLayout>

TIP

常用的属性

android:orientation="vertical"：线性布局的方向 垂直(vertical)/水平(Horizontal) android:id="@+id/ID名称"：控件/布局的ID属性

android:background="@color/colorGreen"：背景颜色，资源的形式表示颜色。 android:background="#f00"：背景颜色，是以#开头的十六进制表示颜色 android:background="#ff0000"：背景颜色，是以#开头的十六进制表示颜色 R G B FF 00 00 00 FF 00 00 00 FF https://www.cnblogs.com/stalla/p/11593566.html

android:layout_width=""：宽度 android:layout_height=""：高度 1.match_parent（匹配父窗口） 2.wrap_content（适应内容） 3.xxxdp(直接写具体的值，单位是dp)

android:text="今天下雨啦啦啦"：显示的文本（一种是使用资源的形式描述，一种是直接卸载布局文件中） android:textSize="20sp"：文字的大小（单位是sp） android:textColor="#00f"：字体的颜色（跟背景颜色的写法是一致的） android:textStyle="bold|italic"：字体风格，（加粗、斜体、正常） android:gravity="bottom|right"：内容的位置“上下左右” android:layout_gravity="bottom":控件在布局中的位置

android:weightSum="10"权重和 android:layout_weight="5"权重比例 如果方向是垂直的，高度设置为0dp。 如果方向是水平的，宽度设置为0dp。

# 使用案例（垂直方向)

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:id="@+id/userInfo"
    android:weightSum="3"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/tvName"
        android:layout_width="match_parent"
        android:layout_height="0dp"
        android:background="#b8be5d"
        android:textSize="20sp"
        android:textColor="#00f"
        android:layout_weight="1"
        android:gravity="center"
        android:text="今天下雨啦啦啦" />
    <TextView
        android:id="@+id/tvAge"
        android:layout_width="match_parent"
        android:layout_height="0dp"
        android:background="@color/colorPrimary"
        android:textSize="20sp"
        android:textColor="#00f"
        android:layout_weight="1"
        android:gravity="center"
        android:text="今天下雨啦啦啦" />
    <TextView
        android:id="@+id/tvSex"
        android:layout_width="match_parent"
        android:layout_height="0dp"
        android:background="@color/colorAccent"
        android:textSize="20sp"
        android:textColor="#00f"
        android:layout_weight="1"
        android:gravity="center"
        android:text="今天下雨啦啦啦" />
</LinearLayout>

# RelativeLayout相对布局

TIP

在相对布局中没有方向这个概念。

默认显示的位置一定是在左上角的（坐标原点的）

第一组：兄弟控件之间的相对位置。属性值是另一个控件的ID。

android:layout_toRightOf="@id/btn2"在..的右边

android:layout_toLeftOf="@id/btn2"在...的左边

android:layout_above="@id/btn2"在..的上边

android:layout_below="@id/btn1"在..的下边

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/btn1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="按钮一" />
    <Button
        android:id="@+id/btn2"
        android:layout_toRightOf="@id/btn1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="按钮二" />
    <Button
        android:id="@+id/btn3"
        android:layout_below="@id/btn1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="按钮三" />
</RelativeLayout>

第二组：兄弟控件之间的对齐关系。属性值是另一个控件的ID。

android:layout_alignRight="@id/btn1"：右边对齐

android:layout_alignLeft="@id/btn1"：左边对齐

android:layout_alignBottom="@id/btn1"：底部对齐

android:layout_alignTop="@id/btn1"：顶部对齐

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/btn1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerInParent="true"
        android:text="按钮一" />

    <Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_toRightOf="@id/btn1"
        android:layout_alignBottom="@id/btn1"
        android:text="按钮二" />
</RelativeLayout>

第三组：指控件与父控件之间的对齐方式，属性是true或者是false。

android:layout_alignParentRight="true"与父控件的右边对齐

TIP

特殊属性：

android:layout_centerInParent="true"正中央居中

android:layout_centerVertical="true"垂直正中央居中

android:layout_centerHorizontal="true"水平正中央居中

TIP

内外边距

控件和控件之间的距离称为外边距。

控件中的内容与控件之间的距离称为内边距。

android:layout_margin="100dp"：外边距（注意四个方向都是有效果的，单位是dp）

android:layout_marginLeft="50dp"：左外边距

android:layout_marginTop="10dp"：上外边距

android:layout_marginBottom="50dp"：下外边距

android:layout_marginRight="100dp"：右外边距

android:padding="10dp"：内边距（注意四个方向都是有效果的，单位是dp）

android:paddingLeft="90dp"：左内边距

android:paddingTop="150dp"：上内边距

android:paddingBottom="100dp"：下内边距

android:paddingRight="100dp"：右内边距

# FrameLayout帧布局

TIP

顺序、大小。 android:foreground="@drawable/computer"：设置帧布局的前景图像

android:foregroundGravity="left|bottom"：设置前景图像的显示位置

<?xml version="1.0" encoding="utf-8"?>
<FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:foreground="@drawable/computer"
    android:layout_height="match_parent"
    android:foregroundGravity="center"
    tools:context=".MainActivity">

    <TextView
        android:layout_width="200dp"
        android:layout_height="200dp"
        android:layout_gravity="center"
        android:background="#f00" />

    <TextView
        android:layout_width="150dp"
        android:layout_height="150dp"
        android:layout_gravity="center"
        android:background="#0f0" />

    <TextView
        android:layout_width="100dp"
        android:layout_height="100dp"
        android:layout_gravity="center"
        android:background="#00f" />

</FrameLayout>
